<h2>How to use</h2>
Well... you can click on this <a src="https://www.jasonuniverse.com.br">link here.</a>
You can also download the portfolio source code with the images and all the code used and execute it localy. There are two ways to do it: By Live Server VSCode Extension OR by runing it localy in a server ( Like Flask if you want python or similar).

<h3>With VSCode</h3>
Just unpack the zip file after download then open it on the vscode root folder, after that, install the live server extension and run it. You can access through your browser localy in the <i>localhost:5500</i> address.

<h3>Without VSCode</h3>
You will need to use a server that suports http or https conections. Execute the server serving the index.html. Access the endpoint that returns the index.html.
